# Installation

```bash
pip install --upgrade --no-cache-dir pre-commit
pip install --upgrade --no-cache-dir flake8
pre-commit install
pre-commit install --hook-type commit-msg
```

# Uninstallation

```bash
pre-commit uninstall
pre-commit uninstall --hook-type commit-msg
```

# How to run

```bash
make migrate
make makemigrations
make runserver
```
